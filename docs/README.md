# Docs Layout

This folder is organized by purpose rather than by file type.

- `artefacts/datasets/`: bundled example or manually annotated datasets kept with the repository
- `artefacts/results/`: curated result artefacts such as aggregated CSVs, benchmark summaries, and evaluation JSON files
- `artefacts/commands/`: helper command files kept for reproducibility or manual reruns

The `docs/` tree is intended to contain submission-facing non-code material only.
